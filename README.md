# CatLocal

CatLocal is a private, local-first iPhone app for turning real cat encounters
into tactile collectible cards.

## Principles

- Photos and collection data stay on device.
- No public map, account, or cloud processing is required.
- Native iOS interactions carry the experience; decoration stays restrained.
- Accessibility and reduced-motion behavior are part of the design.

## Development

Open `CatLocal.xcodeproj` in Xcode 26 and run the `CatLocal` scheme on an
iOS 26 simulator.
